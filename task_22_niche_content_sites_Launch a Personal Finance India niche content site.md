## Site Architecture: The "Pillar-Cluster" Model

The site architecture will follow a simple, intuitive structure:

```
/ (Home) → /budgeting → /budgeting/2024-budget-template
                → /budgeting/budgeting-tips-for-students
                → /budgeting/budget-apps-review-2024
/budgeting/ (pillar) – 2,500 words – target KW: “budget planner India”
```

Repeat this structure for the other four pillars. Each pillar is a hub page (2,500 words) that links to 3-4 cluster posts (800-1,200 words). Clusters capture long-tail traffic and funnel authority to the hub.
